
Este es un proyecto para analizar repositorios de código fuente descargados desde URLs públicas. Utiliza Node.js y varias bibliotecas para manejar la descarga, extracción y análisis de los archivos del repositorio.
## Características
- Descarga repositorios desde URLs públicas.
- Extrae archivos comprimidos.
- Analiza el contenido del repositorio para extraer información relevante.
## Requisitos
- Node.js v14 o superior
## Instalación
1. Clona este repositorio:

    git clone
2. Navega al directorio del proyecto:   

    cd repo_analyzer
3. Instala las dependencias:

    npm install
## Uso
    node index.js 
Sigue las instrucciones en la consola para proporcionar la URL del repositorio que deseas analizar.
